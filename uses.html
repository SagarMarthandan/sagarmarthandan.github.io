Hardware · Software · Stack

What I use

Tools, software, and infrastructure that power my data engineering workflow. Constantly evolving, rarely settled.

01

Hardware

Lenovo ThinkPad X1 Carbon

Daily driver for data engineering. Light, powerful, great keyboard for long coding sessions.

Dell UltraSharp 27" 4K

Secondary monitor for dashboards, monitoring, and documentation.

Logitech MX Master 3S

Ergonomic, precise, silent clicks. The horizontal scroll wheel is a data-wrangling lifesaver.

Keychron K8 Pro

Mechanical keyboard with hot-swappable switches. Gateron Browns for the balance of feel and noise.

02

Data Stack

dbt Core & Cloud

Transformations, testing, documentation — the backbone of every analytics project. Jinja + SQL.

Apache Airflow

Orchestration and scheduling. DAGs for batch pipelines, sensors for event-driven triggers.

Snowflake

Cloud warehouse of choice. Zero-copy cloning, time travel, and separation of compute/storage.

Google BigQuery

Serverless analytics at scale. Columnar storage, BI Engine, and BigLake for lakehouse patterns.

Azure Databricks

Spark-optimized analytics. Delta Lake for ACID transactions, Unity Catalog for governance.

Apache Kafka

Streaming and event processing. Real-time data pipelines for live dashboards and microservices.

Airbyte

Open-source EL(T). Connector catalog for APIs, databases, and file-based ingestion.

Dagster

Asset-based orchestration. Software-defined assets for lineage, freshness, and declarative scheduling.

03

Cloud & Infrastructure

Google Cloud Platform

BigQuery, Composer (Airflow), Cloud Storage, IAM. Primary cloud for analytics workloads.

Microsoft Azure

Databricks, Data Factory, ADLS Gen2, Key Vault, Entra RBAC. Enterprise data platform.

Terraform

Infrastructure as code for cloud resources, IAM policies, and data warehouse configs.

Docker

Containerized pipelines, local dev environments, Airflow workers, and Spark clusters.

GitHub Actions

CI/CD for dbt, data pipeline testing, Terraform plans, and artifact deployment.

GitLab CI

Alternative CI/CD runner. Used for multi-stage pipeline deployments and regression testing.

04

Languages & Tools

Python

Primary language. Pandas, PySpark, Airflow DAGs, FastAPI, LangChain, and scripting.

SQL

PostgreSQL, Snowflake SQL, BigQuery SQL, Spark SQL, DQL/DML/DDL, window functions.

VS Code

Editor of choice. Extensions for dbt, Python, SQL, YAML, and Jupyter notebooks.

Power BI

BI and reporting. DAX, Power Query, row-level security, and paginated reports.

Streamlit

Rapid data apps and LLM demos. Prototyping analytics interfaces for non-technical stakeholders.

Obsidian

Knowledge base and personal wiki. Markdown vault for data engineering notes and reference.
